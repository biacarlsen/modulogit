Comandos Linux Terminal 

Para entrar na raiz do HD(disco local):

cd C:

Para acessar todos os usuarios do sistema:

cd C:/Users

Para acessar um usuario do sistema:

cd C:/Users/bia_c(nome do usuario)

Para ir para área de trabalho:

cd Desktop

Para entrar em uma pasta:

cd nomedapasta 

Para voltar uma pasta:

cd ../ 
 
Para listar tudo de uma pasta:

ls

Para criar uma pasta:  

mkdir nomedapasta

Para iniciar um repositório:

git init 

Para ver as modificações:

git status

Para adicionar um arquivo: 

git add nomedoarquivo.extensão

Para adicionar todos os arquivos que não estão trackeados(que o git nao reconhece):

git add -A

Para limpar a tela:
Clear


O que é Branch?
Versões diferentes de um sistema. A versão principal se chama MASTER.

O que é um Commit?
Serve para enviar as alterações no sistema (sempre usar comentários-msgs-para explicar
as alterações). 

O que é o README.md?
Arquivo onde é colocado informações descritivas importantes para leituras posteriores.


Para dar um Commit:

git commit -m "Mensagem"


Para listar todos os Commits feitos:
git log


Para Reverter Modificações :

dar um git log para ver os commits feitos;
copiar o código da versão(id do commit) que a pessoa quer voltar;
dar um git reset com a especificação desejada.

Tipos de reset:
reset --soft    volta para a versão desejada porem mantem as alterações de forma não commitadas;  

reset --mixed   faz a mesma coisa do soft porém, ele nao vai estar pronto para 
commitar as informações alteradas;

reset --hard    vai ignorar todas as alterações feitas depois da versão escolhida.
